Grilli - Restaurant Website

Grilli is a fully responsive restaurant website, Responsive for all devices, build using HTML, CSS, and JavaScript.

Objectif du Projet

Le principal objectif de ce projet de fin d'études est de concevoir et de développer un site web interactif et attrayant pour un restaurant en utilisant les technologies web modernes telles que HTML5, CSS3 et JavaScript. Le site doit offrir une expérience utilisateur agréable tout en présentant les services, le menu, et les informations essentielles du restaurant de manière claire et intuitive.

Technologies Utilisées

HTML5 :
Utilisé pour structurer le contenu du site web de manière sémantique et accessible.
CSS3 :
Utilisé pour styliser le site web, en créant une mise en page attrayante et en assurant la réactivité pour différents dispositifs (ordinateurs, tablettes, smartphones).
JavaScript :
Utilisé pour ajouter des fonctionnalités interactives et dynamiques, améliorant ainsi l'expérience utilisateur.

Le projet de création d'un site web pour un restaurant a permis de mettre en pratique les compétences acquises en développement web tout en répondant aux besoins réels d'un client. Le résultat est un site moderne, réactif et convivial qui améliore la visibilité en ligne du restaurant et facilite l'interaction avec les clients.
